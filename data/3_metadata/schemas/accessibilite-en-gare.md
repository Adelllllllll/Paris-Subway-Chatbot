# Schéma des données : accessibilite-en-gare.json

Ce document décrit la structure du fichier de données associé.

| Nom du champ | Type | Description |
| :--- | :--- | :--- |
| **accessibility_level_id** | `integer` | Niveau d'accessibilité |
| **accessibility_level_name** | `string` | Légende du niveau d'accessibilité |
| **commentaire** | `string` | Permet d'indiquer si le niveau d'accessibilité n'est pas le même pour les différentes lignes de la gare |
