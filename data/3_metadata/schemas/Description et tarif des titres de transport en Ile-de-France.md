# Schéma des données : Description et tarif des titres de transport en Ile-de-France.json

Ce document décrit la structure du fichier de données associé.

| Nom du champ | Type | Description |
| :--- | :--- | :--- |
| **Nom du Produit** | `string` | Nom complet du produit  |
| **Description** | `string` | Description du produit  |
| **Indication du produit** | `string` | Indication sur le produit |
| **Période du produit** | `string` | Période de validité du produit  |
| **LIEN VERS LE PORTAIL** | `string` | Lien vers le site du titre  |
| **DURÉE** | `string` | Durée d'application du titre |
| **PROFIL** | `string` | Profil du voyageur |
| **Durée en anglais** | `string` | Durée d'application du titre écrit en anglais |
| **Profil en anglais** | `string` | Profil écrit en anglais |
| **Lien recharger produit** | `string` | Lien du bouton pour recharger le produit  |
| **Description image du produit** | `string` | Description de l'image du produit pour les malvoyants |
| **Image du produit** | `string` | L'url de l'mage du produit  |
