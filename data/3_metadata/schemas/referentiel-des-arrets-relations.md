# Schéma des données : referentiel-des-arrets-relations.json

Ce document décrit la structure du fichier de données associé.

| Nom du champ | Type | Description |
| :--- | :--- | :--- |
| **PdEId** | `string` | Identifiant ICAR du pôle d'échanges parent (facultatif) |
| **PdEVersion** | `integer` | Numéro de version du pôle d'échanges parent (facultatif) |
| **ZdCId** | `string` | Identifiant ICAR de la zone de correspondance parente |
| **ZdCVersion** | `integer` | Numéro de version de la zone de correspondance parente |
| **ZdAId** | `string` | Identifiant ICAR de la zone d'arrêts parente |
| **ZdAVersion** | `string` | Numéro de version de la zone d'arrêts parente |
| **ArRId** | `string` | Identifiant ICAR de l'arrêt parent |
| **ArRVersion** | `string` | Numéro de version de l'arrêt parent |
| **ArTId** | `string` | Identifiant ICAR d'un arrêt transporteur  |
| **ArTVersion** | `string` | Numéro de version de l'arrêt transporteur  |
