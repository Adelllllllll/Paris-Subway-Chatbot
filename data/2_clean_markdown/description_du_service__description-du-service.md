Les API Prochains Passages

# Description du service

Mis à jour le 28 novembre 2024

La Plateforme d’échanges Île-de-France Mobilités **récolte** les informations, les **diffuse** aux autres transporteurs, les **met à disposition** de tous.

**Les services Les API Prochains passages offrent des données concernant :**​

* [Les](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[prochains](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[passages](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[pour](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[toutes](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[les](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[lignes](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[disponibles](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[sur](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale) [le](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale)[réseau](https://rec.prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_globale?ref=prim.iledefrance-mobilites.fr)​
* [Les](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire)[prochains](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire)[passages](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire) [à](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire)[un](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire)[arrêt](https://rec.prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire?ref=prim.iledefrance-mobilites.fr)[particulier](https://prim.iledefrance-mobilites.fr/apis/idfm-ivtr-requete_unitaire)​

**Autres données disponibles utiles pour requêter les API Prochains passages :**​

Pour connaitre les données relatives aux référentiels (arrêts et lignes), disponible sur le portail d’Île-de-France Mobilités, suivez [ce](https://prim.iledefrance-mobilites.fr/fr/jeux-de-donnees/perimetre-des-donnees-tr-disponibles-plateforme-idfm?staticDataSlug=perimetre-des-donnees-tr-disponibles-plateforme-idfm) [lien](https://prim.iledefrance-mobilites.fr/fr/jeux-de-donnees/perimetre-des-donnees-tr-disponibles-plateforme-idfm?staticDataSlug=perimetre-des-donnees-tr-disponibles-plateforme-idfm).​

*La liste des données disponibles est mise à jour toutes les semaines.*​​

* **Le portail des normes pour les données d'offre du transport collectif** : [https://www.normes-donnees-tc.org/](https://www.normes-donnees-tc.org/?ref=prim.iledefrance-mobilites.fr)​
* **Profil des données :** le SIRI Lite​
* **Format des horaires :** aaaa-mm-jj ‘T‘ hh:mm:ss.SSSZ​
